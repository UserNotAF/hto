# HTO - Hiddify Torrent Offline

> BitTorrent protection module for Hiddify Manager.

## Philosophy

HTO is designed to protect Hiddify servers from BitTorrent-related abuse reports without affecting normal VPN usage.

### Goals

- ✅ Keep YouTube working
- ✅ Keep Telegram working
- ✅ Keep Discord working
- ✅ Keep websites working
- ✅ Keep games working
- ❌ Block BitTorrent traffic only
